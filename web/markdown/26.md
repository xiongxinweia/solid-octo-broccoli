## nc

Linux nc命令用于设置路由器。

执行本指令可设置路由器的相关参数。

```bash
nc [-hlnruz][-g<网关...>][-G<指向器数目>][-i<延迟秒数>][-o<输出文件>][-p<通信端口>][-s<来源位址>][-v...][-w<超时秒数>][主机名称][通信端口...]
```



## 链接博客地址

```
nc nsddd.top 80  #80是默认的
```

**协议是约定的，一定要按照规范写，否则不认**

```
[root@mail ~]# nc www.baidu.com 80
```
